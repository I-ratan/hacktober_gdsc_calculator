# Hacktober_GDSC_Calculator

Its just for fun nothing else.

## Installation

Use the package manager [pip](#) to install foobar.

```bash
pip install hackcalci
```

## Usage

```python
import calci

# returns 'ratan'
calci.pluralize('ratan')

# returns 'himanshu'
calci.pluralize('himanshu')

# returns 'sandesh'
foobar.singularize('sandesh')
```
